常用布局

布局分为两个部分： 尺寸和定位。布局类似堆积木层层堆叠，通过CSS获取Web页面元素并控制他们的相对位置

普通文档流

不同文档流的排列顺序为：从左到右，从上到下。内容由块元素和行内元素构成。块元素通常由新的一行开始排列，可通过CSS改变它的宽高，一般情况下，一行只能容纳一个块元素。行内元素则是由内容撑开其大小，不会被CSS所设定的宽高来改变大小。可在一行内布置多个行内元素并且通过多个元素构成一行。行内块元素则既可以多个元素一起堆叠为一行又可以通过CSS来改变宽高。
在CSS中可以通过display属性来实现元素的类型转换

布局可以使内容更清晰，使页面加载更快（将布局样式放入一个CSS文件中，减小了HTML文件的代码量。访问页面时可以HTML和CSS文件同时加载。加快了网页的加载效率），SEO优化。




CSS flex布局

flex(flexBox) 弹性布局，flex是一种一维布局模式可以改变我们在一维文档流中的默认布局既（从左到右，从上到下）
采用Flex布局的元素被称为Flex容器，它的子元素自动成为容器成员，称为Flex项目。容器存在两个轴。一个是水平方向的主轴。一个是垂直方向的纵轴。主轴开始位置叫做 main start 结束位置叫做 end start。交叉轴的开始位置叫做cross start 结束位置叫做cross end. 项目占据的主轴空间叫做main size 占据的交叉轴空间叫做cross size。

flex-dirction 属性定义要在哪个方向上堆叠flex项目。

flex-warp 属性定义是否需要对flex项目进行换行

flex-flow 为同时设置flex-dirction和flex-warp的简写属性，用法为

justify-content 属性用于对齐 flex 项目

flex-grow 属性规定某个 flex 项目相对于其余 flex 项目将增长多少

flex-shrink 属性规定某个 flex 项目相对于其余 flex 项目将收缩多少。

flex-basis 属性规定 flex 项目的初始长度。

CSS 定位

定位分为 浮动和定位
定位为把一个元素从它原本的位置移动到到应有的位置
分为static relative absolute fixed 四种属性

static

static 为HTMl元素的默认定位元素，不受top，left等元素的控制。且不会以任何特殊方式定位，它始终根据页面的正常文档流进行定位

relative

relative的元素是根据其正常位置来进行定位的
通过设置top right bottom left 的值来使其偏离正常的位置并以此来进行位置的调整

absolute

position: absolute; 的元素相对于最近的定位祖先元素进行定位（而不是相对于视口定位，如 fixed）。然而，如果绝对定位的元素没有祖先，它将使用文档主体（body），并随页面滚动一起移动
“被定位的”元素是其位置除 static 以外的任何元素。

fixed

position: fixed; 的元素是相对于视口定位的，这意味着即使滚动页面，它也始终位于同一位置。 top、right、bottom 和 left 属性用于定位此元素。固定定位的元素不会在页面中通常应放置的位置上留出空隙。
